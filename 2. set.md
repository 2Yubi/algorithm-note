# set とは

set は固有の要素のみを格納するコンテナ

### 特徴

1. {key, value}だけではなく int 形とか pair でも格納できる
2. 重複した値は除去される
3. map と同じく自動で整列される

```
#include<bits/stdc++.h>
using namespace std;
int main(){
	set<int> int_st;

	int_st.insert(5);
	int_st.insert(5);
	int_st.insert(5);
	int_st.insert(5);
	int_st.insert(1);
	int_st.insert(2);

	for(auto i : int_st) cout << i << " ";

	return 0;
}

// 結果
// 1 2 5

```
